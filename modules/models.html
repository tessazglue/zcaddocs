Django Models¶

Chiplet¶

Database Schema for Chiplet Application, which consists of Chiplet, IO, Config and ConfigIO classes.

class chiplet.models.Chiplet(*args, **kwargs)[source]¶

Stores a chiplet with primary key being the unique CID of chiplet. IO refers to a ball on a chiplet.

Custom model methods:
get_io_count(): returns the number of IOs of that chiplet get_cfg_count(): returns the number of configs of that chiplet get_ios(): returns a QuerySet of IOs of that chiplet get_cfgs(): returns a QuerySet of Configs of that chiplet
class chiplet.models.Config(*args, **kwargs)[source]¶

Stores a Config with foreign key to a chiplet to represent a one-chiplet-to-many-configs relationship.

Custom model methods:
get_cfgios(): returns a QuerySet of ConfigIOs of that Config
class chiplet.models.ConfigIO(*args, **kwargs)[source]¶

Stores a ConfigIO with foreign key to a config to represent a one-config-to-many-configIOs relationship, and foreign key to an IO to represent a one-IO-to-one-configIO relationship.

class chiplet.models.IO(*args, **kwargs)[source]¶

Stores an IO with foreign key to a chiplet to represent a one-chiplet-to-many-IOs relationship.

System¶

Database Schema for System Application, which consists of Sys, SysComponent and SysNetlist classes.

class system.models.Sys(*args, **kwargs)[source]¶

Stores a system and its checks.

Custom model methods:
get_syscomponents(): returns a QuerySet of SysComponents of that system
class system.models.SysComponent(*args, **kwargs)[source]¶

Stores a system component and its packing information. Foreign keys to a chiplet, a config and a sys.

Custom model methods:
get_sysnetlist(): returns a QuerySet of SysNetlist of that SysComponent
class system.models.SysNetlist(*args, **kwargs)[source]¶

Stores a system IO and its information. Foreign keys to a syscomponent and a configIO.
